# Quiz Or Sheet v2 - Project Rules

## 🛠 Tech Stack
- **Frontend**: Next.js 14+ (App Router).
- **Backend**: Python (FastAPI).
- **AI**: Google Gemini (Flash 1.5).
- **Styling**: Vanilla CSS (Premium/Dark/Medical theme).

## 🏗 Architecture
- **Monorepo**: `/frontend` and `/backend`.
- **Agentic Logic**: Always use Python for processing and AI workflows.
- **Real-time**: Use SSE (Server-Sent Events) for live agent status streaming.

## 🔒 Security
- NEVER hardcode API keys.
- Use `.env` in the backend.
- Ensure `.gitignore` is always up to date.

## 🚀 Workflow
- **Plan First**: Always update `docs/plans/` before major changes.
- **Git**: Commit after completing each major task in the `task.md`.
- **Aesthetics**: Every UI component must feel "Premium" (smooth transitions, harmonized colors).
